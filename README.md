# Hücre Laboratuvarı / Cell Laboratory

## Açıklama / Description
Hücre Laboratuvarı, biyolojik yapıları eğlenceli ve etkileşimli öğrenmeni sağlayan bir web uygulamasıdır. Kullanıcı hücreden organizmaya kadar yapıları keşfeder.  
Cell Laboratory is a web app to learn biological structures in a fun and interactive way. Users explore levels from cells to organisms.

## Kullanılan Teknolojiler / Technologies
- JavaScript  
- HTML  
- CSS  

## Projeyi Çalıştırma / How to Run
1. Proje klasörünü indir.  
2. `index.html` dosyasını tarayıcıda aç.  
3. Uygulama çalışacaktır.

## Özellikler / Features
- Etkileşimli öğrenme modülleri  
- Seviye atlama sistemi  
- Görsel içerikler
